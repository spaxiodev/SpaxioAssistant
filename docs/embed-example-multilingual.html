Multilingual widget example

Optional: set SpaxioAssistantConfig before the script to override locale or chatbot ID.

Developer API (after widget loads):

  • SpaxioAssistant.setLanguage('es') – switch widget language
  • SpaxioAssistant.getLanguage() – current language code
  • SpaxioAssistant.onLanguageChange(function(lang) { ... }) – listen for changes
